# BigaData - Compiler Design & Language Processing

A comprehensive collection of compiler design algorithms and language processing tools with easy copy functionality.

## Features

### **Syntax Validators:**
- **Java Switch Statement Validator** - Complex regex-based validation
- **Java Loop Validator** - While, For, Do-While loop validation
- **Java If-Else Validator** - Conditional statement validation
- **Python Syntax Validator** - AST-based syntax checking

### **Code Generation:**
- **Three Address Code Generator** - Triplets & Quadruples
- **Enhanced TAC Generator** - Table format output
- **Mnemonic Code Converter** - TAC to assembly conversion
- **Assembly Code Validator** - Opcode validation

### **Language Processing:**
- **Grammar Production Rules Parser** - Terminal/Non-terminal identification
- **Symbol Table Generator** - Variable tracking with addresses
- **Regular Expression Validators** - Pattern matching examples
- **Finite Automata String Validator** - State transition validation
- **Python Tokenizer** - Keywords & Identifiers extraction
